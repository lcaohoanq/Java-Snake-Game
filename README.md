## Need to install
- JDK 18.0.2.1
- Maven
- Docker, run docker container mysql with:
```bash
docker-compose up -d
```

```bash
# verify connection
mysql -h 127.0.0.1 -P 3307 -u lcaohoanq -p
use snake_game_app;
select * from users;
```
